## Upstream Documentation

The original imgaug documentation is still valuable for learning concepts:

- [Original imgaug Docs](https://imgaug.readthedocs.io/) - API reference and guides
- [imgaug-doc Repository](https://github.com/aleju/imgaug-doc) - Notebooks and examples

!!! note
    When using upstream resources, change imports from `imgaug` to `imgaug2`.
